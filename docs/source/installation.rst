Installation
============

The most recent release can be installed from `PyPI
<https://pypi.org/project/curies_processing>`_ with uv:

.. code-block:: console

    $ uv pip install curies_processing

or with pip:

.. code-block:: console

    $ python3 -m pip install curies_processing

Installing from git
-------------------

The most recent code and data can be installed directly from GitHub with uv:

.. code-block:: console

    $ uv pip install git+https://github.com/cthoyt/curies-processing.git

or with pip:

.. code-block:: console

    $ python3 -m pip install git+https://github.com/cthoyt/curies-processing.git

Installing for development
--------------------------

To install in development mode with uv:

.. code-block:: console

    $ git clone git+https://github.com/cthoyt/curies-processing.git
    $ cd curies-processing
    $ uv pip install -e .

or with pip:

.. code-block:: console

    $ python3 -m pip install -e .
